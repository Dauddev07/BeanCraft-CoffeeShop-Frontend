# ☕ BeanCraft – Coffee Shop Web App

A modern, responsive coffee shop web application built using React and deployed on Vercel. The application provides a seamless user experience for browsing products, managing a cart, and interacting with a clean, visually appealing UI.

---

## 🚀 Live Demo

🔗 https://bean-craft-coffee-shop-frontend.vercel.app/

---

## 📌 Features

* 🏠 Interactive Home Page with Hero Section
* 📋 Dynamic Menu with Product Listings
* 🛒 Add to Cart & Quantity Management
* 🔐 Login & Signup Modal System
* ⚠️ Smart Alerts for User Actions
* 💳 Checkout & Payment Selection UI
* 📦 Order Summary & Success Page
* ⭐ Product Rating System
* 📱 Fully Responsive Design
* 🎨 Smooth UI Animations & Hover Effects

---

## 🛠️ Tech Stack

* **Frontend:** React, JavaScript, HTML5, CSS3
* **Routing:** React Router
* **Build Tool:** Vite
* **Deployment:** Vercel
